# 💫 About Me:
Experimental physicist with a thing for making matter behave, and then simulating it just to be sure.<br>Currently deep in computational physics — building physics simulations from the ground up and advancing in COMSOL Multiphysics and LAMMPS for molecular dynamics and material modeling.<br>I work in C++, Python, and MATLAB, and I leverage Claude AI as a middleware layer for rapid prototyping and intelligent pipeline design — though the architecture, deployment, and final execution are always mine.<br>Physics doesn't scare me on the bench or on the screen — I'm as comfortable running experiments as I am writing code to model them.<br>I build apps and interactive tools because translating complex physical systems into something a human can actually use is its own kind of science.<br>Comfortable with HTML and CSS when the job calls for a clean front-end layer, but the real love is the logic underneath.<br>Currently leveling up in finite element analysis, multiphysics simulation, and atomistic modeling.<br>My background spans nanomaterial synthesis, photocatalysis, and materials characterization — XRD, bandgap analysis, nanocomposite systems.<br>If it involves pushing a physical system to its limits and then writing code to describe why, I'm already interested.<br>Always building. Always experimenting. Occasionally sleeping.
